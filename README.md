# Number Guessing Game 

A simple JavaScript number guessing game where the user tries to guess a randomly generated number.

## Features
- User-defined range
- Random number generation
- Hints for wrong guesses
- Option to quit anytime

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Click **Start Game**
2. Enter the maximum number
3. Guess the number
4. Type `quit` to exit

Open the browser console to see messages.
